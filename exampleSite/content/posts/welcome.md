Welcome to your new Hugo blog powered by **Swiss Operator**.

## Features

This theme includes:

- Clean Swiss design aesthetic
- Dark mode support
- Responsive layout
- Multi-language support (EN, ES, FR)
- Category color coding
- Remark42 comments integration

## Code Example

```go
func main() {
    fmt.Println("Hello, Swiss Operator!")
}
```

## Typography

The theme uses **Inter** for body text and **JetBrains Mono** for code blocks, giving it a clean technical feel.
